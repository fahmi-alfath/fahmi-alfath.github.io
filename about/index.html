Tentang Penulis

Seorang Mahasiswa di salah satu kampus di Sukabumi yang masih menekuni bidang pemrograman terutama pemrograman web dengan bahasa PHP. Terbiasa menggunakan PHP Framework Codeigniter dalam pembuatan aplikasi dan juga website.

Tujuan adanya website ini diantaranya yaitu sebagai portofolio hasil dari yang pernah pelajari dan kerjakan, membagikan pengetahuan seperti cara membuat sesuatu dan juga perbaikan masalah / error pada suatu kode.

Apa saja yang saya kuasai?

Saya menguasai beberapa bahasa pemrograman, diawali dengan bahasa yang biasa dikuasai dalam pemrograman web seperti HTML, CSS murni maupun dengan Framework seperti Bootstrap dan Semantic UI, Javascript dengan menggunakan JQuery, dan PHP native serta menggunakan Codeigniter dan Laravel.

Untuk pemrograman desktop yang pernah saya pelajari dan gunakan yaitu Java Desktop menggunakan Netbeans dan juga C++ yang baru bisa saya gunakan untuk membuat aplikasi penghitungan menggunakan Terminal / CMD.
